# Retail Sales & Customer Insights

## Problem Statement
Analyze retail sales data to identify trends, customer behavior, and key performance indicators.

## Data
- Sales transactions
- Customer information
- Product categories

## Tools
SQL, Python, Power BI

## Key Work
- Designed ETL pipelines to clean and transform raw sales data
- Conducted exploratory data analysis using Python
- Built Power BI dashboards with KPIs

## Outcome
- Identified top-performing categories and regions
- Provided insights to support inventory and marketing decisions
