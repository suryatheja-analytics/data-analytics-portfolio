# Snake Game (Python)

A classic Snake game built using Python and the turtle module to practice object-oriented programming and game logic.

## Concepts Used
- Object-Oriented Programming
- Class-based design
- Game loop and state management
- Collision detection
- Event handling (keyboard input)
- File handling for score persistence

## Tools
- Python
- turtle module
